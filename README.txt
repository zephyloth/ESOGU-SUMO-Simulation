1-) ESOGU-PROJE Dosyasını Sumo/bin klasörünün içine sürükleyiniz.
2-) baslat.py dosyasını çalıştırınız. (Örneğin: cmd'de python baslat.py)
3-) Açılan SUMO uygulamasında F9 tuşuna basıp "View settings" kısmından gerekli ayarları yükleyiniz. (Ayar.xml)
4-) Artık simülasyon hazır. "Run" butonuna basabilirsiniz.
